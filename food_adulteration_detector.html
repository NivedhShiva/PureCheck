How it works MobileNetV2 · TFLite
01
Image Input
Camera frame or uploaded image resized to 224×224 px and normalized to float32 [0–1].
02
Feature Extraction
MobileNetV2 pretrained on ImageNet extracts deep visual features via depthwise convolutions.
03
Classification
Dense layers classify the sample as Pure or Adulterated using softmax probability output.
04
TFLite Deploy
Model is quantized and exported as .tflite for fast on-device inference on Android/iOS.
